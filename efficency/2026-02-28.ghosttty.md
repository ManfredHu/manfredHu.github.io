# Ghostty
是一款与iTerm2类似的终端工具，支持跨系统。因为使用GPU渲染，性能比[iTerm2](https://iterm2.com/)更优，是一款更现代的终端工具

倡导约定配置定功能，而不是像iTerm2一样提供各种插件。所以默认化集成的能力已经很丰富了，与zsh结合看起来也非常顺畅。本文默认你已经是iTerm2深度用户，重点在于解释如何平滑切换iTerm2到ghostty，以及一些iTerm2没有的功能

一些你可能感兴趣的疑问

1. 如何类似iTerm2一样全局快捷键唤起？
2. 如果做多窗口管理？
3. 如何在终端显示图片？iTerm2是做不到在终端显示图片的，而ghostty是支持的，甚至可以直接在终端显示gif图片，感觉非常炫酷

## 常用命令

```bash
# 列出所有内置主题（200+ 个）
ghostty +list-themes

# 列出系统可用字体
ghostty +list-fonts

# 查看所有配置项的默认值和文档
ghostty +show-config --default --docs
```

## 配置

**配置文件**: 按 `Cmd + ,`  就能直接打开配置文件。配置文件位置 `~/.config/ghostty/config`，配置文件使用YAML格式
**重载配置**: 快捷键 `Cmd + Shift + ,`

## 如何类似iTerm2一样快捷键唤起？

这是一个对我来说非常重要的功能，毕竟我习惯了用快捷键来唤起终端工具
而iTerm2是有全局快捷键的, 这里可以用Thor软件设置一个软件呼起快捷键，来唤起ghosttty，就可以实现类似iTerm2一样的快捷键唤起了

## 如果做多窗口管理？

概念上不一致，iTerm2是基于标签页的，而ghosttty是基于工作区的，工作区可以包含多个标签页，每个标签页可以包含多个终端窗口，这样就可以实现多窗口管理了。

比如可以鼠标右键将一个标签页分离(split)为多个工作区，一个工作区对应一个代码库。这样一个标签页就可以集合多个终端窗口了，感觉非常方便。

而不同的项目可以用不同的标签页控制，并行时就不会混乱

1. 分屏(同Tab内)
   - 左右分屏：`Cmd + D`
   - 上下分屏：`Cmd + Shift + D`
   - 放大/还原当前分屏：`Cmd + Shift + Enter`
   - 缩放分屏（增大/减小）：`Cmd + Ctrl + = / -`
   - 关闭当前分屏：`Cmd + W`
2. Tab
   - 下一个Tab：`Cmd + Shift + ]`
   - 上一个Tab：`Cmd + Shift + [`
   - 新建 Tab：`Cmd + T`
   - 切换 Tab：`Cmd + 数字键`
   - 全屏切换：`Cmd + Enter`

```bash

```

## Quick Terminal
Quick Terminal 是 Ghostty 的特色功能，也就是临时终端。可以按照如下配置

```bash
# Quick Terminal
# 全局热键（默认就有，这里可以自定义）
keybind = global:cmd+escape=toggle_quick_terminal
# 下拉终端的位置：top / bottom / left / right / center
quick-terminal-position = top
# 占屏幕比例（v1.2+ 支持精确尺寸）
quick-terminal-size = 80%
# 在哪个屏幕显示：main / mouse / macos-menu-bar
quick-terminal-screen = main
# 自动隐藏：失去焦点时收起
quick-terminal-autohide = true
# 动画时长（秒，0 = 无动画）
quick-terminal-animation-duration = 0
```


## 最终配置

```bash
# ==================== my config ====================

# 亮色用 Catppuccin Latte，暗色用 Catppuccin Mocha, https://catppuccin.com/

theme = light: Catppuccin Latte, dark: Catppuccin Mocha

# 背景透明度（1.0 = 完全不透明，0.85 = 微透明）

background-opacity = 0.85

# 隐藏原生标题栏，获得更多屏幕空间

# macos-titlebar-style = hidden

# 默认的 Cmd+Shift+[/] 是循环切换分屏。想按方向跳（像 Vim 那样）可以加：

keybind = cmd+shift+h=goto_split:left
keybind = cmd+shift+j=goto_split:down
keybind = cmd+shift+k=goto_split:up
keybind = cmd+shift+l=goto_split:right

# Quick Terminal

# 全局热键（默认就有，这里可以自定义）

# keybind = global:cmd+escape=toggle_quick_terminal

# 下拉终端的位置：top / bottom / left / right / center
quick-terminal-position = top

# 占屏幕比例（v1.2+ 支持精确尺寸）

quick-terminal-size = 80%

# 在哪个屏幕显示：main / mouse / macos-menu-bar

quick-terminal-screen = main

# 自动隐藏：失去焦点时收起

quick-terminal-autohide = true

# 动画时长（秒，0 = 无动画）

quick-terminal-animation-duration = 0

# 永远记住窗口状态（分屏布局、Tab、目录）

window-save-state = always

# 新分屏/Tab 继承当前目录

window-inherit-working-directory = true

# 新窗口继承字体大小
window-inherit-font-size = true

# 内边距（像素），让文字不贴边

window-padding-x = 4
window-padding-y = 4

# 缩小窗口时，内边距等比缩小

window-padding-balance = true

# 类似 iterm2 的快捷键设置，允许用户通过快捷键打开一个快速终端窗口，配合 Thor 已实现
```
