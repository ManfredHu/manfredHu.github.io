# yazi

看名字你应该懂的，作者肯定是个中国人。逼着大家学中文，官网 https://yazi-rs.github.io/

工具特点

* 文件预览
* 文件复制，剪切，删除，移动
* 代码高亮浏览（自带）
* 选择文件后Enter自动编辑器打开（默认Vim，怎么修改成自己喜欢的编辑器？）
  * 可以根据[这里](https://yazi-rs.github.io/docs/configuration/yazi#opener)的设置修改打开器为自己需要的方式，比如`.ts`用VSCode打开，`.md`用Typora打开，其他的用Vim打开

## 安装

homebrew 安装请看[这里](https://yazi-rs.github.io/docs/installation#homebrew)

除了yazi你还会连带着安装 ffmpeg，7-Zip等常用的一堆实用工具，比如ffmpeg经常处理音视频文件转化格式等，非常好用

## Vim

yazi默认与Vim实用习惯强关联，也就是hjkl四个键，你可以完成文件夹打开，后退(hl)，选择文件的操作(jk)

同理箭头的上下左右跟jkhl键盘是一一对应的，如果不习惯vim用箭头也可以操作

## 文件查找

使用 `fzf` 查找文件, `fzf` 是一个命令行模糊查找工具，yazi集成了它来实现快速文件查找功能。可以在yazi中按 `z` 键进入文件查找模式，输入关键词进行模糊搜索后过滤，选择文件后回车就跳转到文件目录了

## 文件操作

### 复制&粘贴

你可以使用类似vim的 `y` 在选择文件后进行复制， `p` 就是粘贴，而且会自动重命名

### 剪切&粘贴

你可以使用类似vim的 `x` 在选择文件后进行剪切， `p` 就是粘贴，而且会自动重命名

比如使用空格选择文件后按 `x` 就是剪切了，之后在目标目录按 `p` 就会粘贴了

### 文件删除

选择文件后按 `d` 就可以删除了，删除的文件会被放到回收站里，可以在回收站里恢复

### 文件重命名

选择文件后按 `r` 就可以在弹出的对话框中进行重命名，输入新的名字后回车即可

特别说的一点是，弹出的对话框也支持vim的快捷键，比如 `Ctrl + a` 可以快速移动到行首， `Ctrl + e` 可以快速移动到行尾，非常方便

## 路径记录

可以先安装 `zoxide`，这是一个命令行工具，可以记录你访问过的路径，并且根据访问频率进行排序，之后在yazi中按 `Z` 键就可以进入路径记录模式，输入关键词进行模糊搜索后过滤，选择路径后回车就跳转到该路径了

可以手动将常用的路径添加到zoxide的DB里, 也可以把autojump的历史记录导入到zoxide里

```sh
zoxide add [path]
```

### 设置别名
比如设置`z=zoxide`, 把 `eval "$(zoxide init bash)"` 记到 `~/.bashrc` 里即可

其他方式可以在[这里](https://github.com/ajeetdsouza/zoxide?tab=readme-ov-file#installation) 自取

### 导入autojump历史记录

比如导入默认的autojump历史记录到zoxide里可以执行

```sh
zoxide import --from=autojump "$HOME/Library/autojump/autojump.txt"
```

## 命令行格式化json

复制JSON字符串到剪贴板后，终端输入 `pbpaste | jq .` 就可以格式化JSON输出了

比如测试一个数据 `curl xxxx | jq .` 也可以将返回格式化输出到终端里
