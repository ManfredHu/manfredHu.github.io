# ghosttty
是一款与iterm2类似的终端工具，支持跨系统。性能比itemr2更优

## 配置
倡导约定配置定功能，而不是像iterm2一样提供各种插件。所以默认化集成的能力已经很丰富了，与zsh结合看起来也非常顺畅


1. 如何类似iterm2一样快捷键唤起？
2. 如果做多窗口管理？
3. 如何在终端显示图片？iterm2是做不到在终端显示图片的，而ghosttty是支持的，甚至可以直接在终端显示gif图片，感觉非常炫酷

## 如何类似iterm2一样快捷键唤起？
这是一个对我来说非常重要的功能，毕竟我习惯了用快捷键来唤起终端工具
而iterm2是有全局快捷键的, 这里可以用Thor软件设置一个软件呼起快捷键，来唤起ghosttty，就可以实现类似iterm2一样的快捷键唤起了

## 如果做多窗口管理？
概念上不一致，iterm2是基于标签页的，而ghosttty是基于工作区的，工作区可以包含多个标签页，每个标签页可以包含多个终端窗口，这样就可以实现多窗口管理了。

比如可以鼠标右键将一个标签页分离(split)为多个工作区，一个工作区对应一个代码库。这样一个标签页就可以集合多个终端窗口了，感觉非常方便。

而不同的项目可以用不同的标签页控制，并行时就不会混乱

## 透明度背景
配置项：https://ghostty.org/docs/config/reference#background-opacity

比如配置如下可以实现类似iterm2的效果

```
background-opacity = 0.85
```
