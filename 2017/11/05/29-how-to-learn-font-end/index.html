IT行业这么火，怎么入坑前端？

2017-11-05 随笔

作者：ManfredHu
链接：http://www.manfredhu.com/2017/11/05/29-how-to-learn-font-end/index.html
声明：版权所有，转载请保留本段信息，谢谢大家

IT行业工资高啊，前端简单啊，妹子都可以搞，那汉子肯定也可以吧。
前端工资高啊，比测试和运维高了一截。而且好火啊，我们去报名培训班吧。
然后肯定就出现一大堆搞前端开发的刚入门的同学了。
我们这篇文章会从一个比较快速的方法入坑前端，会告诉你要看什么书做什么事，书大家可以去京东或者当当买来看，也看网上的PDF版本，要去哪里找？我知道你懂得！！

一般的同学，没有方法会去参加培训班。然后培训两个月后培训的老师会教你怎么刷面经，怎么装逼，然后你照着做你就找到工作了。前端工作其实很好找，因为现在很缺前端，当然更缺的是专业的前端开发人员。

一直觉得培训班对前端的定义非常狭隘，没有培养人家钻研的精神、解决问题的方法，反而一上来就框架、语法，怎么能出东西怎么来。然后就工作了，对一些HTML标签的正确使用、CSS的布局、JS闭包的概念懂个大概或者是完全不懂。看懂了一些代码，然后网上查一查谷歌百度搜一下就CP大法过来直接用。毕竟天朝百度还是非常容易的，代码也多，查一下Copy大法用起来，功能就完成了。可能大家都用过脚本大全这个网站？

好了废话不多说，我们一步一步来，建议收藏本篇文章，方便大家后面找来查看。
